# Мы ищем неотрицательные целые числа, состоящие не более чем из трех цифр, например, сумма кубиков их цифр и есть само число; назовем их «кубическими» числами.
# 153 is such a "cubic" number : 1^3 + 5^3 + 3^3 = 153
# Эти «кубические» числа, состоящие не более чем из трех цифр, легко найти даже вручную, поэтому они «скрыты» с другими числами и символами в строке.

# Задача состоит в том, чтобы найти или нет «кубические» числа в строке, а затем составить сумму этих «кубических» чисел, найденных в строке, если они есть, и вернуть такую ​​строку, как:
# "number1 number2 (и тд при необходимости) sumOfCubicNumbers Lucky"

# если были найдены "кубические" числа число1, число2, ....
# Числа в выходных данных должны быть в том порядке, в котором они встречаются во входной строке.
# Если кубических чисел не найдено, верните строку: `" Unlucky "` `.

# Пример:
#  - s = "aqdf&0#1xyz!22[153(777.777"
#    группы, состоящие не более чем из 3 цифр: 0 и 1 (одна цифра), 22 (две цифры), 153, 777, 777 (3 цифры)
#    Только 0, 1, 153 кубические, а их сумма равна 154
#    Return: "0 1 153 154 Lucky"

# - s = "QK29a45[&erui9026315"
#   Група чисел 29, 45, 902, 631, 5. Не сумма кубов.
#   Return: "Unlucky"

# Примечания
# В строке «001234», где 3 или более цифр следуют друг за другом, первая группа для проверки - «001», а следующая - «234». Если был принят пакет, состоящий не более чем из трех цифр, независимо от того, является ли он «кубическим», для этого пакета все закончено.
# Когда непрерывная строка цифр превышает 3, строка разбивается на группы по 3 слева. Последняя группировка может состоять из 3, 2 или 1 цифры.
# например, «24172410» превращается в 3 строки, состоящие из «241», «724» и «10»
# например, «0785» превращается в две строки, состоящие из «078» и «5».
